Douyin E-commerce User Feature Analysis

Project Overview
This project analyzes user characteristics of Douyin e-commerce using Python. It focuses on user age, gender, and consumption behavior distribution, and provides simple data insights through charts and visualization.

Dependencies
- Python 3
- pandas
- matplotlib
- Jupyter Notebook

How to Run
1. Open the `.ipynb` file in Jupyter Notebook
2. Run all cells to view data and charts
3. All charts are generated based on the full dataset

Main Findings
Age Users range from 18 to 64, with the highest activity among middle-aged users around 40-45, showing broad appeal across different age groups.
Spending: Most users make low-value purchases, with only a small number of moderate and high spenders. The distribution is heavily skewed toward lower spending amounts.
